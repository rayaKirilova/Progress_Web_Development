Описание Тестов линк CSS
Следния линк показва ефекта от :link
(след рефреш на страницата линка ще бъде показан като непосетен) 

          a:link{
            color: green;
          }
Щракнете върху линка и задръжте левия бутон, за да видите ефекта от :active Активиране на връзка 

          a:active{
            font-size: 2em;
            color: red;
          }
Посетете линка за да видите ефекта от :visited Посетена връзка 

          a:visited{ 
            color: gray; 
          }
Позиционирайте мишката върху линка за да видите ефекта от :hover Посочване на връзка 

          a:hover{
             display: inline-block;      
             padding: 3px;
             text-decoration: underline;
             color: #FFF;
             background-color: #C9460F;
             border: 5px outset #FF6000; 
          }
Щракнете върху линка за да видите ефекта от :focus Фокус върху връзка 

          a:focus{
            font-size: 1.5em;
            color: orange;
          }
